# Run (Streaming-with-auto-prompter)

## Install

```bash
cd "/Users/sachin/Documents/New project/odyssey-simulation/Streaming-with-auto-prompter"
npm install
```

## Start

```bash
npm run dev
```

Vite will print a local URL. Open it in the browser.

## Notes
- This project depends on the Odyssey client SDK. You will need a valid Odyssey API key in the UI when prompted.
